Versão inicial do código fonte

Observações:

* As listagens deste livro foram escritas para a versão 3.1 do Python, 
  as instruções para instalação se encontram no capítulo 2 do livro.

* Você precisa de um editor de textos com suporte a edição em UTF-8
  Windows: PSPad, Notepad++, IDLE
  Linux: vim, joe, gedit, emacs, IDLE
  Mac OS X: TextMate, vim, IDLE

* Algumas listagens devem ser digitadas ou copiadas no interpretador Python.
  Estas listagens tiveram os símbolos de >>> e ... removidos de forma
  a facilitar este trabalho. Lembre-se que o interpretador imprime o retorno
  das funções na tela e que você precisa utilizar a função print caso deseje
  o mesmo em um programa.
  
* Caso encontre erros, favor enviar um email para erros@nilo.pro.br

= Teste do suporte a UTF-8 ===================================================

aeiou com acento agudo: áéíóú
aeiou com acento grave: àèìòù
ao com til: ãõ
aeiou com acento circunflexo: âêîôû
aeiou com trema: äëïöü
cedilha: ç

As linhas acima possuem caracteres acentuados. Se você não visualizar 
corretamente estes acentos, verifique se seu editor de textos tem suporte a
UTF-8.
